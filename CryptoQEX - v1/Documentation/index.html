CryptoQEX Template Documentation

Thank You for purchasing this template, I hope You enjoy it!

1. Template Structure

  • - You can find all files in CryptoQEX folder
  • - In CryptoQEX folder you will see CryptoQEX - color folders
  • - CSS, js, images files you can find in assets folder
  • - CSS files you can find in assets/css folder
  • - JS files you can find in assets/js folder
  • - Images you can find in assets/img folder

2. How to edit template

  • - Change images. Upload your image to assets/img/ folder. Open index.html file and find img tag. In src="http://placehold.it/1980x700/333" change src value to your image path. F.e. src="assets/img/image.jpg"
  • - All texts you can change in .html files

    • - How to edit countdown

    • Open file assets/js/main.js
    • Find and change May 10,2022 00:00 to date which you need
